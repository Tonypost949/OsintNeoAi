## Installation

```bash
# In Termux:
bash install.sh

# Or manually:
pkg install python python-pip sqlite
pip install rich prompt-toolkit pygame
python src/retro_osint_gui.py
```

## Controls

| Action | GUI Mode | Terminal Mode |
|--------|----------|---------------|
| Select entity | Click card | Type ID (e.g., PER-003) |
| View network | Click "Check connections" | `network PER-003` |
| Export | Click "Export to Maltego" | `export` |
| Smoking guns | Click red button | `redflags` |
| Back | Click "<- BACK" or ESC | N/A |

## Features

### Visual Mode (Pygame)
- 8-bit entity icons with glow effects for red flags
- Particle explosions for smoking gun discoveries
- Screen shake on critical finds
- Semi-transparent suggestion buttons
- Scanline CRT effect

### Terminal Mode (Fallback)
- Rich text formatting with colors
- Interactive prompts
- Same database and export functionality

### Both Modes
- Master OSINT Sheet entity structure
- Risk scoring (0-100)
- Level progression
- Persistent SQLite database
- Maltego .mtgl export
- Commentary engine with 30+ quips

## Export to Maltego

Click "Export to Maltego" or type `export`:
- Creates `exports/investigation.mtgl` (graph file)
- Creates `exports/osint_config.mtz` (config file)
- Import into Maltego: File вҶ’ Import вҶ’ Graph from XML
