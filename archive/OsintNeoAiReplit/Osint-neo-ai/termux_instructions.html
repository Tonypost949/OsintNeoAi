ANDROID PHONE SCANNER — TERMUX SETUP GUIDE
STEP 1 OF 4
Install Termux on your Android phone

Download from F-Droid (not Play Store — the F-Droid version has full storage access):

https://f-droid.org/packages/com.termux/
⚠️ Do not use the Play Store version — it has restricted storage access and the scanner will not work.
STEP 2 OF 4
Run setup in Termux (one time only)

Open Termux and paste this entire block:

pkg update -y && pkg install -y python python-pip && pip install openpyxl Pillow mutagen && termux-setup-storage
When asked "Allow Termux to access files?" — tap Allow.
STEP 3 OF 4
Download the scanner to your phone

In Termux, run:

curl -o ~/phone_scanner.py https://REPLACE_WITH_YOUR_REPLIT_URL/phone_scanner.py
Or manually copy phone_scanner.py from this project into your Downloads folder, then in Termux run:

cp ~/storage/downloads/phone_scanner.py ~/phone_scanner.py
STEP 4 OF 4
Run the scanner
python ~/phone_scanner.py

You will see this menu:

1. Documents
2. DCIM (Camera Photos)
3. Downloads
4. WhatsApp
5. Contacts (VCF)
6. All phone storage
7. Custom path

Pick a number, set max files, and it scans. Results saved to:

/storage/emulated/0/OSINT_Neo/OSINT_Master.xlsx
HOW THE MASTER LIST GROWS
Run it as many times as you want

Every scan adds only new files to the master.
Files already scanned are automatically skipped — no duplicates.
The master Excel file has 4 sheets:

📊 Summary    — Total counts and scan history log
📁 All Files   — Every file ever scanned with metadata
📍 GPS Data   — Only files with embedded GPS coordinates
👤 Contacts   — VCF contact files (name, phone, email)
